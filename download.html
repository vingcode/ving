Installation Instructions

Linux

  1. Download vingc file
  2. Open terminal in the folder with the file
  3. Make file executable:
    chmod +x vingc
  4. Test it works:
    ./vingc
  5. Done! Now you can compile .ving files

Windows

  1. Download vingc.exe file
  2. Open command prompt (Win+R, type cmd)
  3. Navigate to folder with file:
    cd path\to\folder\with\vingc.exe
  4. Test it works:
    vingc.exe
  5. Done! Now you can compile .ving files

Build from Source

If you want to build the compiler yourself:

  1. Clone the repository: git clone https://github.com/vingcode/ving.git
  2. Install GCC compiler
  3. Compile:
    # Linux make # Windows (requires MinGW) make windows

See BUILD.md for details.
